# Aptura Desktop Integration

`aptura-desktop` keeps the default experience close to GNOME while adding
Aptura-specific defaults and practical tools.

Included integration points:

- Aptura System Check for local update, firmware, power, security, and disk
  status.
- GNOME favorites and appearance defaults from `aptura-settings`.
- Firmware, power profile, disk, backup, and software management packages.
- Installer launcher integration for the live session.
